SimpleID PEAR channel

Registering the channel: 
pear channel-discover simpleid.org
Listing available packages: 
pear remote-list -c simpleid
Installing a package: 
pear install simpleid/package_name
Installing a specific version/stability: 
pear install simpleid/package_name-1.0.0
pear install simpleid/package_name-beta
Receiving updates via a feed: 
http://simpleid.org/pear/feed.xml
